# 삼성전자 재무제표 분석 챗봇

삼성전자의 재무제표 및 영업 실적 발표자료 PDF를 업로드하여 AI 챗봇과 대화할 수 있는 Streamlit 애플리케이션입니다.

## 시스템 요구사항

- Python 3.8 이상 (개발환경 Python 3.11.9)
- OpenAI API 키
- 충분한 저장 공간 (PDF 처리 및 데이터베이스용)

## 설치 방법

### 1. 저장소 클론 또는 파일 다운로드

```bash
# 파일들을 원하는 디렉토리에 다운로드하세요
```

### 2. 가상 환경 생성 (권장)

```bash
python -m venv your-venv-name
```

# Windows
your-venv-name\\Scripts\\activate

### 3. 필수 라이브러리 설치

```bash
pip install -r requirements.txt
```

### 4. 환경 변수 설정

새파일을 생성한 뒤 '.env'로 제목을 지음.

그 다음 OPENAI_API_KEY=api-key 넣어서 환경변수 설정


## 실행 방법

```bash
streamlit run main.py
```

브라우저에서 `http://localhost:8501`로 접속하세요.

## 사용법

### 1. PDF 업로드
- 사이드바에서 삼성전자 재무제표 PDF 파일을 업로드 (이미 업로드 되어있으면 건너뛰어도 됨)
- 자동으로 분석이 시작되며 완료까지 기다려주세요

### 2. 채팅 시작
- "새 채팅 시작" 버튼을 클릭하여 새로운 채팅 세션 생성
- 또는 기존 채팅 목록에서 이전 대화 선택

### 3. 질문하기
- 채팅 입력창에 재무제표 관련 질문 입력
- AI가 업로드된 문서를 바탕으로 답변 제공
- 답변에는 참고한 문서의 출처가 표시됩니다

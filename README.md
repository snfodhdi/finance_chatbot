# Finance Chatbot
### 2025.11.06 by.VIVA

## About
Finance RAG Chatbot based on Python. <br>
Reads image based PDF file of SAMSUNG ELECTRONICS financial statements. <br>
Reranks 5 most related statements, and makes answer based on reranked statements. <br>
Answer must be scored 0 to 100 using RAGAS, if the answer score is lower than 70, chatbot makes new better answer. <br>
Needs OPENAI API KEY to run.
